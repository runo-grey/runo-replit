# Runo Economy Bot

A Discord economy & gambling bot using Runos currency.

## Commands
All commands work with `!` prefix and `/` slash commands.

**Economy:** balance, daily, work, deposit, withdraw, give
**Gambling:** slots, coinflip, blackjack, roulette, dice, scratch, rob
**Shop:** shop, buy, inventory
**Other:** leaderboard, help
**Admin:** /gamesetup channel

## Setup
1. Copy `.env.example` to `.env` and fill in values
2. `npm install`
3. `npm run dev`
